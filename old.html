Languages turned to linguistics which turned to grammer which turned to algorithms and computer programming

At first I just learned about algorithms in C++ using books and various internet sites before moving onto html/css/js so I could start making stuff. I learned a bit about frameworks with FreeCodeCamp using ReactJS.

I then scrubbed up on the foundations in computer science by completing Harvard's CS50x on EdX, this was where I learned about Flask and Python which I loved. After a bit of research I taught myself Django which is where I'm at now.
